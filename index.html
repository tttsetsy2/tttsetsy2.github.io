To begin

  1. Insert your key in the Key tab

Decrypting Your Save

  1. Run save_editor.py
  2. Open the savedata.txt file located at: %appdata%\Spooky House Studios UG (haftungsbeschr√§nkt)\Progressbar95\Documents\savedata.txt
  3. The program will create a file named decrypted_save.txt

Editing Your Save

  1. Open decrypted_save.txt with any text editor
  2. Make your desired changes to the save data
  3. Save and close the file

Re-encrypting Your Save

  1. Run the save editor again
  2. Click the Encrypt button
  3. Copy %appdata%\Spooky House Studios UG (haftungsbeschraenkt)\Progressbar95 and paste it inside of the Run (Win+R) box
  4. Copy and paste the Documents folder as a backup
  5. Enter the Documents folder and, if you backed the Documents folder, delete the original savedata.txt
  6. Copy the encrypted_save.txt here and rename it to savedata.txt
  7. Launch Progressbar95 to play with your modified save
